# I-Topi-Mangiano-Gatti
Download For I Topi Mangiano Gatti

Until further notice, in order to run this game you MUST have a file named C:\Users\Public\Documents\user.txt, containing your username!

1. Download or clone this repository
2. Open file explorer
3. Navigate to location you downloaded this repository to
4. Double click on Mice-Eat-Cats-Setup.msi. This will open the installation wizard
5. Go through the wizard. READ EVERYTHING! (If you don't know what something is, leave it as it is.)
6. Make sure the user.txt file is in the correct location as stated above and contains your username. THIS IS CASE SENSATIVE
7. Either navigate to your desktop and click on I Topi Mangiano Gatti or the windows program menu and search for I Topi Mangiano Gatti
8. The program should start after this
9. While in the game, press escape to exit
