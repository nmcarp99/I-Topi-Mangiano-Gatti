# I-Topi-Mangiano-Gatti
Download For I Topi Mangiano Gatti

1. Download or clone this repository
2. Open file explorer
3. Navigate to location you downloaded this repository to
4. Double click on Mice-Eat-Cats-Setup.msi. This will open the installation wizard
5. Go through the wizard. READ EVERYTHING! (If you don't know what something is, leave it as it is.)
6. Either navigate to your desktop and click on I Topi Mangiano Gatti or the windows program menu and search for I Topi Mangiano Gatti
7. The program should start after this
8. While in the game, press escape to exit
